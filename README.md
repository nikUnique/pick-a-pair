# Pick-a-Pair

A simple memory-matching card game built with **TypeScript**.

## About

Pick-a-Pair is a classic memory game where you flip cards to find matching pairs. Test your memory and try to match all pairs in as few moves as possible!

## How to Play

- Click any card to reveal its hidden value
- Click another card to try to find its match
- If the two cards match, they stay flipped and you score a point
- If they don't match, both cards flip back over
- **Win condition**: Match all 8 pairs to complete the game
- **Track your progress** with move counter, match counter, and timer

## Features

✨ Simple, intuitive gameplay  
⏱️ Real-time move and match tracking  
⏰ Game timer  
🔄 Reset button to start a new game

## Installation

1.  Clone this repository:

    ```bash
    git clone <repository-url>
    cd pick-a-pair
    ```

2.  Install dependencies:

    ```bash
    npm install
    ```

3.  Start the development server:

    ```bash
    npm start
    ```

4.  Open your browser and navigate to the local server address

## Technology Stack

- TypeScript – Type-safe JavaScript
- HTML5 – Game structure
- CSS3 – Styling and animations
